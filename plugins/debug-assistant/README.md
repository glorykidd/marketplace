# debug-assistant

Interactive debugging toolkit with structured root cause analysis and an autonomous investigation agent.

## Usage

```
/debug-assistant:analyze "TypeError: Cannot read property 'id' of undefined"
@debug-assistant:investigate  (as an agent)
```

## Commands

| Command | Description |
|---|---|
| `analyze` | Structured root cause analysis with ranked fix candidates |

## Agents

| Agent | Description |
|---|---|
| `investigate` | Autonomous end-to-end bug investigation — traces code, runs searches, proposes a fix |
