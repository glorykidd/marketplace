# commit-message

Generates clear, conventional commit messages from your staged git changes.

## Usage

```
/commit-message:generate
/commit-message:generate auth
```

## Commands

| Command | Description |
|---|---|
| `generate` | Generate a commit message from staged changes |

## Output

Produces a [Conventional Commits](https://www.conventionalcommits.org/) formatted message with type, optional scope, subject, and body.
