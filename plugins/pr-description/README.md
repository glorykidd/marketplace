# pr-description

Generates detailed, reviewer-friendly pull request descriptions from your branch diff.

## Usage

```
/pr-description:generate
/pr-description:generate develop
```

## Commands

| Command | Description |
|---|---|
| `generate` | Generate a PR description diffed against main (or specified base branch) |

## Output

Produces a complete PR description with Summary, Changes, Test Plan, and Notes sections.
